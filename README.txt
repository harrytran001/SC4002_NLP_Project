1. Pre-requisites:
Python 3.10: Download from https://www.python.org/downloads/

2. Running code with VS Code
2.1. Python environment:
2.1.1 Mac
- Create environment:
cd path/to/your/project
python3 -m venv myenv
- Activate environment:
source myenv/bin/activate

2.1.2 Windows
- Create environment:
cd path/to/your/project
python -m venv myenv
- Activate environment:
myenv\Scripts\activate
or
.\myenv\Scripts\Activate

2.2. Run code
- Open each .ipynb file
- Choose the environment created as the kernel
- Run all cells

3. Code File Structure
- Final - P1,2 - GloVe and Final - P1,2 - Word2vec: Run GloVe and Word2vec with the same settings to select Word2vec for the next experiment
- Final - P2 - Word2vec - Config: Experiment with hyperparameters for Word2vec
- Final - P3_Q1 - Word2vec: Response for P3_Q1
- Final - P3_Q3 - Word2Vec: Response for P3_Q3
- Final - P3_Q4 - Word2Vec: Response for P3_Q4
- Final - P3_Q5 - Word2Vec: Response for P3_Q5
